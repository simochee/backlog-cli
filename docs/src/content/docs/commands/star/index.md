スターの追加・一覧表示を行うコマンド群です。

## サブコマンド

| コマンド                                        | 説明                   |
| ----------------------------------------------- | ---------------------- |
| [star add](/commands/star/add/)     | スターを追加する       |
| [star list](/commands/star/list/)   | スターの一覧を表示する |
| [star count](/commands/star/count/) | スター数を表示する     |
