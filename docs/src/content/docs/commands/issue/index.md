Backlog の課題を管理するコマンド群です。

## サブコマンド

| コマンド                                              | 説明                           |
| ----------------------------------------------------- | ------------------------------ |
| [issue list](/commands/issue/list/)       | 課題の一覧を表示する           |
| [issue view](/commands/issue/view/)       | 課題の詳細を表示する           |
| [issue create](/commands/issue/create/)   | 課題を作成する                 |
| [issue edit](/commands/issue/edit/)       | 課題を編集する                 |
| [issue close](/commands/issue/close/)     | 課題を完了にする               |
| [issue reopen](/commands/issue/reopen/)   | 課題を再オープンする           |
| [issue comment](/commands/issue/comment/) | 課題にコメントを追加する       |
| [issue status](/commands/issue/status/)   | 自分の課題ステータスを表示する |
