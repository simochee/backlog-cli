---
title: backlog status-type list
description: ステータスの一覧を表示する
---

```
backlog status-type list [flags]
```

## オプション

| フラグ      | 短縮 | 型     | 説明                                       |
| ----------- | ---- | ------ | ------------------------------------------ |
| `--project` | `-p` | string | プロジェクトキー（env: `BACKLOG_PROJECT`） |

## 使用例

```bash
backlog status-type list --project PROJ
```
