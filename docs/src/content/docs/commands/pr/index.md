Backlog Git のプルリクエストを管理するコマンド群です。

## サブコマンド

| コマンド                                          | 説明                                   |
| ------------------------------------------------- | -------------------------------------- |
| [pr list](/commands/pr/list/)         | プルリクエストの一覧を表示する         |
| [pr view](/commands/pr/view/)         | プルリクエストの詳細を表示する         |
| [pr create](/commands/pr/create/)     | プルリクエストを作成する               |
| [pr edit](/commands/pr/edit/)         | プルリクエストを編集する               |
| [pr close](/commands/pr/close/)       | プルリクエストをクローズする           |
| [pr merge](/commands/pr/merge/)       | プルリクエストをマージする             |
| [pr reopen](/commands/pr/reopen/)     | プルリクエストを再オープンする         |
| [pr comment](/commands/pr/comment/)   | プルリクエストにコメントを追加する     |
| [pr comments](/commands/pr/comments/) | プルリクエストのコメント一覧を表示する |
| [pr status](/commands/pr/status/)     | 自分のプルリクエストの状態を表示する   |
