Backlogスペースの認証を管理するコマンド群です。

## サブコマンド

| コマンド                                | 説明                             |
| --------------------------------------- | -------------------------------- |
| [auth login](/commands/auth/login/)     | Backlog スペースに認証する       |
| [auth logout](/commands/auth/logout/)   | 認証情報を削除する               |
| [auth status](/commands/auth/status/)   | 認証状態を表示する               |
| [auth token](/commands/auth/token/)     | 認証トークンを標準出力に表示する |
| [auth refresh](/commands/auth/refresh/) | OAuth トークンをリフレッシュする |
| [auth switch](/commands/auth/switch/)   | アクティブなスペースを切り替える |
