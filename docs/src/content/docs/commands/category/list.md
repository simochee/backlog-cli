---
title: backlog category list
description: カテゴリの一覧を表示する
---

```
backlog category list [flags]
```

対応する Backlog API については「[カテゴリー一覧の取得](https://developer.nulab.com/ja/docs/backlog/api/2/get-category-list/)」を参照してください。

## オプション

`-p`, `--project <string>`
: プロジェクトキー（env: `BACKLOG_PROJECT`）

## 使用例

```bash
backlog category list --project PROJ
```
