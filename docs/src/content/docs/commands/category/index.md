## サブコマンド

| コマンド                                                  | 説明                     |
| --------------------------------------------------------- | ------------------------ |
| [category list](/commands/category/list/)     | カテゴリの一覧を表示する |
| [category create](/commands/category/create/) | カテゴリを作成する       |
| [category edit](/commands/category/edit/)     | カテゴリを編集する       |
| [category delete](/commands/category/delete/) | カテゴリを削除する       |
