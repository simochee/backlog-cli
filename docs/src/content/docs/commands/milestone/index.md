プロジェクトのマイルストーンを管理するコマンド群です。

## サブコマンド

| コマンド                                        | 説明                           |
| ----------------------------------------------- | ------------------------------ |
| [milestone list](/commands/milestone/list/)     | マイルストーンの一覧を表示する |
| [milestone create](/commands/milestone/create/) | マイルストーンを作成する       |
| [milestone edit](/commands/milestone/edit/)     | マイルストーンを編集する       |
| [milestone delete](/commands/milestone/delete/) | マイルストーンを削除する       |
