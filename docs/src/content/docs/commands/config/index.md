---
title: backlog config
description: CLI の設定を管理します
---

CLIの設定を管理するコマンド群です。設定は `~/.backlogrc` ファイルに保存されます。

## サブコマンド

| コマンド                              | 説明                       |
| ------------------------------------- | -------------------------- |
| [config get](/commands/config/get/)   | 設定値を取得する           |
| [config set](/commands/config/set/)   | 設定値を変更する           |
| [config list](/commands/config/list/) | すべての設定を一覧表示する |
