---
title: backlog watching read
description: ウォッチを既読にする
---

```
backlog watching read <watching-id>
```

対応するBacklog APIについては「[ウォッチの既読化](https://developer.nulab.com/ja/docs/backlog/api/2/mark-watching-as-read/)」を参照してください。

## 引数

`<watching-id> <int>`
: ウォッチID

## 使用例

```bash
backlog watching read 12345
```
