---
title: backlog watching view
description: ウォッチの詳細を表示する
---

```
backlog watching view <watching-id>
```

対応するBacklog APIについては「[ウォッチ情報の取得](https://developer.nulab.com/ja/docs/backlog/api/2/get-watching/)」を参照してください。

## 引数

`<watching-id> <int>`
: ウォッチID

## 使用例

```bash
backlog watching view 12345
```
