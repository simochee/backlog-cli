---
title: backlog project users
description: プロジェクトメンバーを表示する
---

```
backlog project users <project-key>
```

プロジェクトに所属するメンバーの一覧を表示します。

対応するBacklog APIについては「[プロジェクトユーザー一覧の取得](https://developer.nulab.com/ja/docs/backlog/api/2/get-project-user-list/)」を参照してください。

## 引数

`<project-key> <string>`
: プロジェクトキー

## 使用例

```bash
backlog project users PROJ
```
