Backlog のプロジェクトを管理するコマンド群です。

## サブコマンド

| コマンド                                                          | 説明                               |
| ----------------------------------------------------------------- | ---------------------------------- |
| [project list](/commands/project/list/)               | プロジェクトの一覧を表示する       |
| [project view](/commands/project/view/)               | プロジェクトの詳細を表示する       |
| [project create](/commands/project/create/)           | プロジェクトを作成する             |
| [project edit](/commands/project/edit/)               | プロジェクトを編集する             |
| [project delete](/commands/project/delete/)           | プロジェクトを削除する             |
| [project users](/commands/project/users/)             | プロジェクトメンバーを表示する     |
| [project add-user](/commands/project/add-user/)       | プロジェクトにメンバーを追加する   |
| [project remove-user](/commands/project/remove-user/) | プロジェクトからメンバーを削除する |
| [project activities](/commands/project/activities/)   | プロジェクトの最近の更新を表示する |
