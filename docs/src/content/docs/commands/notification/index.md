---

Backlog の通知を管理するコマンド群です。

## サブコマンド

| コマンド | 説明 |
|----------|------|
| [notification list](/backlog-cli/commands/notification/list/) | 通知の一覧を表示する |
| [notification count](/backlog-cli/commands/notification/count/) | 未読通知数を表示する |
| [notification read](/backlog-cli/commands/notification/read/) | 通知を既読にする |
| [notification read-all](/backlog-cli/commands/notification/read-all/) | すべての通知を既読にする |
