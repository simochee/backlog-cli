---
title: backlog notification read
description: 通知を既読にする
---

```
backlog notification read <id>
```

指定した通知を既読にします。

対応する Backlog API については「[お知らせの既読化](https://developer.nulab.com/ja/docs/backlog/api/2/read-notification/)」を参照してください。

## 引数

`<id> <int>`
: 通知 ID

## 使用例

```bash
backlog notification read 12345
```
