---
title: backlog issue-type list
description: 課題種別の一覧を表示する
---

```
backlog issue-type list [flags]
```

## オプション

| フラグ      | 短縮 | 型     | 説明                                       |
| ----------- | ---- | ------ | ------------------------------------------ |
| `--project` | `-p` | string | プロジェクトキー（env: `BACKLOG_PROJECT`） |

## 使用例

```bash
backlog issue-type list --project PROJ
```
