プロジェクトの課題種別を管理するコマンド群です。

## サブコマンド

| コマンド                                                      | 説明                     |
| ------------------------------------------------------------- | ------------------------ |
| [issue-type list](/commands/issue-type/list/)     | 課題種別の一覧を表示する |
| [issue-type create](/commands/issue-type/create/) | 課題種別を作成する       |
| [issue-type edit](/commands/issue-type/edit/)     | 課題種別を編集する       |
| [issue-type delete](/commands/issue-type/delete/) | 課題種別を削除する       |
