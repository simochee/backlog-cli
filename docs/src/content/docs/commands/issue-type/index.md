プロジェクトの課題種別を管理するコマンド群です。

## サブコマンド

| コマンド | 説明 |
|----------|------|
| [issue-type list](/backlog-cli/commands/issue-type/list/) | 課題種別の一覧を表示する |
| [issue-type create](/backlog-cli/commands/issue-type/create/) | 課題種別を作成する |
| [issue-type edit](/backlog-cli/commands/issue-type/edit/) | 課題種別を編集する |
| [issue-type delete](/backlog-cli/commands/issue-type/delete/) | 課題種別を削除する |
