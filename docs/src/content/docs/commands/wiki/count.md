---
title: backlog wiki count
description: Wiki ページ数を表示する
---

```
backlog wiki count [flags]
```

プロジェクトの Wiki ページ数を表示します。

## オプション

| フラグ | 短縮 | 型 | 説明 |
|--------|------|------|------|
| `--project` | `-p` | string | プロジェクトキー（env: `BACKLOG_PROJECT`） |

## 使用例

```bash
backlog wiki count --project PROJ
```
