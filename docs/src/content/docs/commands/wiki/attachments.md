---
title: backlog wiki attachments
description: Wiki ページの添付ファイル一覧を表示する
---

```
backlog wiki attachments <wiki-id>
```

## 引数

| 引数        | 型     | 必須 | 説明      |
| ----------- | ------ | ---- | --------- |
| `<wiki-id>` | number | Yes  | ページ ID |

## 使用例

```bash
backlog wiki attachments 12345
```
