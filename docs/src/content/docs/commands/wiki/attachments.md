---
title: backlog wiki attachments
description: Wiki ページの添付ファイル一覧を表示する
---

```
backlog wiki attachments <wiki-id>
```

対応する Backlog API については「[Wiki添付ファイル一覧の取得](https://developer.nulab.com/ja/docs/backlog/api/2/get-list-of-wiki-attachments/)」を参照してください。

## 引数

`<wiki-id> <int>`
: ページ ID

## 使用例

```bash
backlog wiki attachments 12345
```
