Backlog スペースの情報を管理するコマンド群です。

## サブコマンド

| コマンド | 説明 |
|----------|------|
| [space info](/backlog-cli/commands/space/info/) | スペース情報を表示する |
| [space activities](/backlog-cli/commands/space/activities/) | スペースのアクティビティを表示する |
| [space disk-usage](/backlog-cli/commands/space/disk-usage/) | ディスク使用量を表示する |
| [space notification](/backlog-cli/commands/space/notification/) | スペース通知を表示する |
