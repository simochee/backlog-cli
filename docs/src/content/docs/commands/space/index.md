Backlog スペースの情報を管理するコマンド群です。

## サブコマンド

| コマンド                                            | 説明                               |
| --------------------------------------------------- | ---------------------------------- |
| [space info](/commands/space/info/)                 | スペース情報を表示する             |
| [space activities](/commands/space/activities/)     | スペースのアクティビティを表示する |
| [space disk-usage](/commands/space/disk-usage/)     | ディスク使用量を表示する           |
| [space notification](/commands/space/notification/) | スペース通知を表示する             |
